# jeeves-infra

Infrastructure layer for Jeeves - adapters above the kernel.

## Installation

```bash
pip install -e .
```

## Optional Dependencies

- `gateway`: FastAPI, gRPC, WebSocket adapters
- `postgres`: PostgreSQL and pgvector support
- `redis`: Redis client support
- `embeddings`: Sentence transformers
- `llm`: LiteLLM and HTTP providers
- `all`: All optional dependencies
